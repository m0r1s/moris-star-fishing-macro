# moris Star Fishing Macro
This is a simple macro for the roblox game star fishing.

**Join the discord:** https://discord.gg/2fraBuhe3m

---

## Features

- **Fully Automated Casting**: Automatically casts the Rod with a Perfect throw
- **Fully Automated Reeling**: Instantly Reels in Caught stars
- **Auto-Sell System**: Configurable auto-selling after pre-set amount of cycles
- **Cycle Tracking**: Persistent cycle counter with GUI display (resets per reload)
- **Window Move & Resize**: Auto-resizes and positions Roblox window

---

## Getting Started

### What You Need

- Windows PC
- [Autohotkey V2.0](https://www.autohotkey.com/)
- moris star fishing macro.ahk
- Roblox running

### Controls

| Key | What it does |
|-----|--------------|
| `F1` | Start the macro |
| `F2` | Reload the macro |
| `F3` | Toggle Autosell |

### Getting Started

1. Open Roblox and join Star Fishing
2. Run the script
3. Hit `F1` to start
4. The script will resize your Roblox window and start doing its thing

---

## If Something Goes Wrong

- Make sure Roblox is actually open and you can see the game window
- Check that AutoHotkey v2.0 installed correctly
- Make sure windows scaling setting is set to 100%
- Roblox should be in windowed mode, not fullscreen (the script will try to fix this automatically)
- Join the Discord and ask for Help
